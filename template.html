Welcome to the new site
January 20, 2010

As is obvious before you, this is a new site. In case you were curious, the biggest changes were to the backend (we're now hosted on Google's App Engine in Python as opposed to on standard hosting with PHP), but we've made a number of changes to the front-end as well (which you can see)!

First and foremost is the new home page. Not only does it still display the currently-playing information, but it also shows new adds in an attractive little box there on the left. It also displays the top-played songs and albums of the past X days. And finally, a blog! Or at least something like a blog. We're debating putting up mp3s as it's not clear how best to do it legally.
